# 🎹 Virtual Piano

A responsive web-based Virtual Piano application that allows users to play piano notes directly from their keyboard. The project provides an interactive piano interface with volume and pitch controls, creating a fun and engaging musical experience.

## 🚀 Features

* 🎵 Play piano notes using keyboard keys
* 🎹 Interactive piano key interface
* 🔊 Adjustable volume control
* ⬆️⬇️ Pitch increase/decrease functionality
* 🎼 Octave display and management
* 💻 Responsive design for different screen sizes
* ⚡ Real-time audio playback

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## ▶️ Getting Started

1. Clone the repository

```bash

```

2. Navigate to the project directory

```bash
cd virtual-piano
```

3. Open `index.html` in your browser.

No additional installation is required.
